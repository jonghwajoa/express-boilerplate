# express-boilerplate

## Quick start

```
git clone https://github.com/jonghwajoa/express-boilerplate <YOUR_PROJECT_NAME>

cd <YOUR_PROJECT_NAME>

rm -rf .git

git init

git remote add origin <YOUR_REMOTE_URL>

npm install

npm install -g sass prettier nodemon @commitlint/cli @commitlint/config-conventional

npm start

http://localhost:3000/
```
